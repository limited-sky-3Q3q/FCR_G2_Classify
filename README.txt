FCR轨迹机器学习分析 - 使用说明
========================================

项目简介：
基于互联网的正念减压疗法对于乳腺癌患者癌症复发恐惧心理（FCR）轨迹的
机器学习二次分析（已移除FCR7_0变量）。

========================================
快速开始
========================================

1. 查看分析结果：
   打开 "移除FCR7_0后的分析报告.md"

2. 查看快速摘要：
   python Quick_Summary.py

3. 查看详细报告：
   - 综合分析报告.txt（机器学习分析）
   - 详细统计报告.txt（统计分析）

========================================
主要结果
========================================

【最佳模型】
KNN（K近邻）
- 准确率：82.73%
- F1-Score：0.771
- 参数：n_neighbors=9, metric='euclidean', weights='uniform'

【Top 5 重要特征】
1. GAD7_0（焦虑）- Cohen's d = -1.046（大效应）
2. TCSQ_NC（消极应对）- Cohen's d = -0.972（大效应）
3. Emotional_support（情感支持）- Cohen's d = -0.872（大效应）
4. PHQ9_0（抑郁）- Cohen's d = -0.861（大效应）
5. Knowingduration_of_cancer - Cohen's d = 0.720（中效应）

【类别特征对比】
类别1（高恐惧-快速缓解，n=18）：
- 基线心理症状更严重（焦虑、抑郁、失眠）
- 更多消极应对方式
- 情感支持更强
- 对干预反应快速

类别2（中度恐惧-渐进缓解，n=34）：
- 基线症状较轻
- 癌症知晓时间更长
- 症状缓解较慢

========================================
