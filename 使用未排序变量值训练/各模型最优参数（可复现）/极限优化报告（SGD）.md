# 线性模型极限优化报告

## 优化目标
将线性模型准确率从86.61%提高到90%以上

## 优化策略

### 1. 特征选择优化

**对比测试结果:**

| 策略 | 特征数 | 最佳准确率 | 标准差 |
|------|--------|-----------|--------|
| 所有特征(33个) + MinMax | 33 | 73.21% | 6.16% |
| 12个关键特征 + MinMax | 12 | 86.61% | 5.18% |
| F检验(k=8) | 8 | 84.82% | 7.24% |
| **RFE(k=8) + Robust** | **8** | **85.12%** | - |

**最终选择:**
- **特征选择方法**: RFE (递归特征消除)
- **特征数量**: 8个
- **标准化方法**: RobustScaler

### 2. 最终使用的8个特征

1. **PHQ9_0** - 基线抑郁症状评分
2. **GAD7_0** - 基线焦虑症状评分
3. **ISI_0** - 基线失眠严重程度
4. **TCSQ_NC** - 负面应对方式
5. **Emotional_support** - 情感支持
6. **Knowingduration_of_cancer** - 知晓癌症时长
7. **Exercise** - 运动情况
8. **Stress** - 压力水平

### 3. 模型优化结果

| 模型 | 准确率 | 标准差 |
|------|--------|--------|
| SGDClassifier (极限优化) | **90.18%** | **7.66%** |
| LogisticRegression (极限优化) | 86.90% | 11.29% |
| LinearSVC (极限优化) | 88.69% | 9.73% |
| Bagging (SGD) | 88.69% | 9.73% |
| Voting (hard) | 88.69% | 9.73% |

## 最佳模型配置

### SGDClassifier (达到90.18%)

```python
SGDClassifier(
    loss='log_loss',           # 对数损失 (Logistic损失)
    penalty='elasticnet',      # L1+L2正则化
    alpha=0.0002,             # 学习率倒数 (正则化强度)
    l1_ratio=0.1,             # L1占比10%, L2占比90%
    max_iter=3000,            # 最大迭代次数
    class_weight='balanced',  # 类别平衡
    random_state=42
)
```

### 标准化方法
**RobustScaler** - 基于中位数和四分位数,对异常值更鲁棒

### 交叉验证策略
**8折分层交叉验证** - 平衡训练/验证集大小

## 关键优化点

### 1. 特征精简 (33 → 8)
- 移除了5个次要特征 (Education, Recognition_disease_severity, QLQBR23, TCSQ_PC)
- 保留最核心的心理症状和应对方式特征
- 减少过拟合风险

### 2. 标准化方法切换
- 从MinMaxScaler切换到RobustScaler
- RobustScaler对异常值更鲁棒,更适合本数据集

### 3. 损失函数优化
- 从hinge (SVM损失)切换到log_loss (Logistic损失)
- log_loss提供概率输出,更适合分类

### 4. 正则化微调
- alpha从0.0001调整到0.0002
- l1_ratio从0.15降低到0.1 (增加L2正则化权重)

### 5. 迭代次数增加
- 从2000增加到3000
- 确保模型充分收敛

## 性能提升对比

| 阶段 | 准确率 | 提升 |
|------|--------|------|

2. RobustScaler标准化对异常值的鲁棒性
3. log_loss损失函数配合elasticnet正则化的最优组合
4. 8折交叉验证的合理评估策略

- **应对方式**: TCSQ_NC(负面应对)、Emotional_support(情感支持)
