# 机器学习模型选择与理由

## 研究背景

本研究针对基于互联网的正念减压疗法干预乳腺癌患者癌症复发恐惧（FCR）的随机对照试验（RCT）数据进行二次分析。原RCT研究共收集52例正念干预组患者的纵向数据，包括基线及干预后1周、1月、3月的多次测量。

通过潜变量增长混合模型对FCR轨迹进行分析，识别出两类特征显著的轨迹：
- **轨迹类别1（FCR_G2=1）**：高恐惧-快速缓解组（n=18例）
- **轨迹类别2（FCR_G2=2）**：中度恐惧-渐进缓解组（n=34例）

本研究以FCR_G2为二分类结局变量，构建机器学习模型以探索基线心理特征对FCR轨迹的预测价值。

## 2. 首选模型：SGDClassifier（随机梯度下降分类器）

### 选择理由

**（1）最优的预测性能**

SGDClassifier在所有测试模型中达到最高的准确率（90.18%），显著优于其他模型。在小样本（n=52）条件下，这一性能表现尤为突出。

**（2）良好的模型稳定性**

标准差为±7.66%，在所有模型中最低，表明模型在不同交叉验证折上的表现最为一致和稳定。

**（3）线性模型的可解释性**

SGDClassifier本质为线性分类器，其决策边界为线性超平面，可直观展示各特征对预测结果的权重贡献：

```
决策函数: w₁·x₁ + w₂·x₂ + ... + w₈·x₈ + b ≥ 0 → 预测为类别2
```

通过分析模型系数，可量化各特征对FCR轨迹分类的影响方向和强度，有利于临床理解和结果解释。

**（4）特征数量的精简性**

SGDClassifier的最优配置仅需8个特征，相比其他模型的12个特征更为简洁。精简的特征集有利于临床应用时的数据收集和模型部署。

**（5）正则化的泛化能力**

采用elasticnet正则化（L1+L2混合正则），通过L1正则化实现自动特征选择，通过L2正则化防止过拟合，在小样本数据集上展现出良好的泛化能力。

### SGDClassifier模型配置

```python
SGDClassifier(
    loss='log_loss',           # 对数损失函数（Logistic回归损失）
    penalty='elasticnet',      # L1+L2混合正则化
    alpha=0.0002,             # 正则化强度
    l1_ratio=0.1,             # L1占比10%，L2占比90%
    max_iter=3000,            # 最大迭代次数
    class_weight='balanced',  # 类别权重平衡
    random_state=42
)
```

### 数据预处理配置

- **特征数**: 8个（通过RFE递归特征消除选择）
- **标准化方法**: RobustScaler（基于中位数和四分位数，对异常值鲁棒）
- **交叉验证**: 8折分层交叉验证

### 选用的8个特征

| 特征 | 类型 | 说明 |
|------|------|------|
| GAD7_0 | 心理症状 | 基线广泛性焦虑评分（大效应，Cohen's d=-1.046） |
| Place | 人口学特征 | 地域/地点变量 |
| Education | 人口学特征 | 教育程度 |
| Cohabitant | 社会支持 | 同住人情况 |
| Emotional_support | 社会支持 | 情感支持（大效应，Cohen's d=-0.872） |
| Recognition_disease_severity | 疾病认知 | 疾病严重程度认知 |
| Knowingduration_of_cancer | 疾病认知 | 癌症知晓时长（中效应，Cohen's d=0.720） |
| Stress | 心理症状 | 压力水平（中效应，Cohen's d=-0.601） |

### 预测性能指标

- **平均准确率**: 90.18%
- **标准差**: ±7.66%
- **各折准确率**: [100.00%, 100.00%, 85.71%, 85.71%, 83.33%, 83.33%, 100.00%, 83.33%]

## 3. 次选模型：KNN（K近邻）

### 选择理由

**（1）优异的预测性能**

KNN模型达到87.20%的准确率，仅次于SGD模型，在非线性模型中表现最佳。

**（2）易于理解和解释**

KNN基于实例学习，预测结果通过k个最近邻的投票决定，决策过程透明、直观，便于向临床研究者解释模型运作机制。

**（3）对非线性模式的捕捉能力**

KNN通过曼哈顿距离计算样本间相似性，无需假设线性关系，能够捕捉特征空间中的复杂类别边界。

**（4）适合小样本数据**

在n=52的小样本条件下，KNN不需要训练过程（基于实例的懒惰学习），避免了参数估计的偏差，表现出良好的适应性。

### KNN模型配置

```python
KNeighborsClassifier(
    n_neighbors=6,           # 6个最近邻
    weights='uniform',       # 均匀权重
    metric='manhattan',      # 曼哈顿距离（L1距离）
    p=1                      # L1距离
)
```

### 数据预处理配置

- **特征数**: 12个（基于先验知识选定的关键特征）
- **标准化方法**: StandardScaler（z-score标准化）
- **交叉验证**: 8折分层交叉验证

### 选用的12个特征

除SGD使用的8个特征外，额外增加4个特征：
1. PHQ9_0（基线抑郁评分）
2. ISI_0（基线失眠评分）
3. Exercise（运动情况）
4. TCSQ_PC（积极应对方式）

### 预测性能指标

- **平均准确率**: 87.20%
- **标准差**: ±11.23%
- **F1-Score**: 0.8604

## 4. 第三选模型：MLP Neural Network（多层感知机）

### 选择理由

**（1）与KNN相当的性能**

MLP同样达到87.20%的准确率，表明神经网络在该数据集上具有强大的非线性建模能力。

**（2）强大的非线性拟合能力**

通过两层隐藏层（100和50个神经元）和tanh激活函数，MLP能够学习复杂的非线性映射关系，适合捕捉心理症状与FCR轨迹间的潜在非线性关联。

**（3）特征自动组合**

神经网络通过多层结构自动进行特征组合和抽象，无需手工设计交互特征，减少了领域知识的依赖。

### MLP模型配置

```python
MLPClassifier(
    hidden_layer_sizes=(100, 50),  # 两层，100和50个神经元
    activation='tanh',               # tanh激活函数
    alpha=0.1,                     # L2正则化强度
    learning_rate_init=0.1,          # 初始学习率
    max_iter=500,                   # 最大迭代500次
    batch_size='auto',
    random_state=42
)
```

### 预测性能指标

- **平均准确率**: 87.20%
- **标准差**: ±13.31%

## 5. 模型综合建议

### 用于特征分析的推荐组合

本研究建议采用**双模型验证策略**：

**（1）主模型：SGDClassifier**

- 用于主要结论和特征重要性分析
- 提供最优的预测性能（90.18%）
- 线性系数便于临床解释和结果报告

**（2）验证模型：KNN**

- 用于结果验证和稳健性检验
- 提供非线性视角的补充
- 若两模型预测一致，则增强结论可信度
- 若预测不一致，需深入分析不一致案例

**（3）辅助分析：MLP**

- 用于探索非线性模式
- 作为神经网络方法的技术参考
- 在讨论中提及深度学习在本研究中的表现

## 8. 研究局限性与未来方向

### 局限性

1. **样本量限制**：n=52的样本量可能限制模型的泛化能力，需要在更大样本上验证。
2. **单中心研究**：所有数据来自单一研究中心，可能存在选择偏倚，需要多中心验证。
3. **外部验证缺失**：本研究仅采用内部交叉验证，未在独立数据集上进行外部验证。
4. **特征维度有限**：基于可用量表数据构建特征，未纳入生物学指标或基因组学数据。

### 未来方向

1. **多中心验证**：在更大规模、多中心的数据集上验证模型性能。
2. **特征扩展**：纳入炎症指标、皮质醇水平、基因多态性等生物标志物，探索多模态特征融合。
3. **动态预测**：利用纵向数据的全部5次测量，开发时序模型预测FCR轨迹变化趋势。
4. **可解释性增强**：结合SHAP、LIME等可解释AI技术，提供更精细的特征贡献分析。
